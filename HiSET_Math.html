🟣 IV. Algebraic Concepts · 45% of the test
Solving linear equations — work step by step

Every linear equation follows the same logic: undo what was done to x in reverse order. Get x alone.

  • Distribute first (remove parentheses)
  • Combine like terms on each side
  • Move variable terms to one side using addition/subtraction
  • Divide (or multiply) both sides to isolate x
  • Check: plug your answer back into the original equation
Quadratic equations — 3 methods to know
  • Factoring: works when the quadratic factors cleanly. Set each factor = 0.
  • Quadratic formula: x = [−b ± √(b²−4ac)] / 2a — always works. MEMORIZE THIS (not on formula sheet)
  • Square roots: if no "bx" term: x² = k → x = ±√k
  • The discriminant (b²−4ac) tells you: positive = 2 real solutions, zero = 1 solution, negative = no real solutions
Functions — key ideas
  • A function gives exactly ONE output for each input
  • f(3) means plug 3 into the function — substitute 3 for x
  • Slope = rise ÷ run = (y₂−y₁) / (x₂−x₁)
  • y = mx + b: m is slope, b is y-intercept (where line crosses y-axis)
  • To find x-intercept: set y = 0 and solve
Systems of equations
  • Substitution: solve one equation for a variable, plug into the other
  • Elimination: add or subtract equations to cancel a variable
  • Parallel lines → 0 solutions; same line → infinite solutions; crossing lines → 1 solution
Inequalities
  • Solve like an equation — but FLIP the inequality sign when multiplying or dividing by a negative number
  • Graph on a number line: open circle = doesn't include the value; closed circle = includes it
🟡 I. Numbers & Operations · 19% of the test
Fractions, decimals, and percents — convert freely
  • Percent → decimal: divide by 100 (35% = 0.35)
  • Decimal → percent: multiply by 100 (0.07 = 7%)
  • Fraction → decimal: divide top by bottom (3/4 = 0.75)
  • Percent of a number: multiply (15% of 80 = 0.15 × 80 = 12)
Ratios and proportions
  • Set up as a fraction: a/b = c/d → cross-multiply: ad = bc
  • Always check your units — label everything
  • Percent change = (new − old) / old × 100
Scientific notation
  • Format: a × 10ⁿ where 1 ≤ a < 10
  • Positive exponent → large number (move decimal right)
  • Negative exponent → small number (move decimal left)
  • To multiply: multiply the a values, add the exponents
  • To divide: divide the a values, subtract the exponents
Rational vs. irrational numbers
  • Rational: can be written as a fraction (1/2, 0.75, −3)
  • Irrational: cannot (π, √2, √3)
  • √ of a perfect square is rational: √25 = 5 ✓
  • Rational + rational = rational; rational × irrational = irrational
🔵 II. Measurement & Geometry · 18% of the test
Area and perimeter — label everything
  • Rectangle: Area = l × w, Perimeter = 2l + 2w
  • Triangle: Area = ½ × base × height
  • Circle: Area = πr², Circumference = 2πr (use π ≈ 3.14)
  • Always check: are you solving for area (square units) or perimeter (linear units)?
Pythagorean Theorem — MEMORIZE (not on formula sheet)
  • a² + b² = c² (c is always the longest side — the hypotenuse)
  • To find hypotenuse: c = √(a² + b²)
  • To find a leg: a = √(c² − b²)
  • Common triples to memorize: 3-4-5, 5-12-13, 8-15-17
Angles
  • Supplementary angles add up to 180°
  • Complementary angles add up to 90°
  • Vertical angles are equal (opposite angles formed by two crossing lines)
  • Triangle interior angles always sum to 180°
Volume and surface area
  • Rectangular prism: V = l × w × h
  • Cylinder: V = πr²h
  • These formulas ARE on the provided formula sheet
🟢 III. Data, Probability & Statistics · 18% of the test
Mean, median, mode
  • Mean (average): add all values ÷ how many values
  • Median: middle value when data is ordered from least to greatest
  • Mode: the value that appears most often
  • Outliers affect the mean more than the median — if data has an extreme value, median is often more useful
Probability
  • P(event) = favorable outcomes ÷ total possible outcomes
  • Probability is always between 0 and 1
  • P(A and B) for independent events: multiply the probabilities
  • P(A or B): add probabilities, subtract overlap
Reading graphs and charts
  • Always read the title, axis labels, and units FIRST
  • Scatter plot with upward trend = positive correlation
  • Line of best fit: use it to predict values — slope means rate of change
  • Box plot: shows minimum, Q1, median, Q3, maximum
